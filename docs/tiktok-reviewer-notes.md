# TikTok Reviewer Notes

## App name
Nicaraguan Homes For Rent

## Short description
A rental property website for Nicaragua that lets authorized administrators export creator-owned listing and property-tour videos to an authorized TikTok account.

## Submit-for-review text
We use TikTok's Content Posting API to let authorized administrators of Nicaraguan Homes For Rent export creator-owned property and listing videos from our website workflow to an authorized TikTok account as drafts for final review and posting inside TikTok. This reduces repetitive manual download and re-upload steps while keeping the final publication decision with the authorized TikTok account owner.

## Optional reviewer note
The integration is used by authorized website administrators to distribute original property-tour and rental listing videos associated with live listings on our website. The initial launch requests only draft-upload functionality, not direct public posting.

## Protected admin demo flow
1. Log in to `/admin/login` as an authorized administrator.
2. Choose an existing property from the property list.
3. Open the TikTok Draft Uploads panel.
4. Click `Connect TikTok` and authorize an allowed TikTok account.
5. Upload a local property-tour video or reuse a stored listing video.
6. Click `Upload Draft to TikTok`.
7. Confirm the status changes and follow the prompt to finish inside TikTok.

## Launch posture
- Business website, not personal use.
- Authorized admins only.
- Original listing media only.
- Draft upload only for phase 1.
- No direct public publishing in this release.
